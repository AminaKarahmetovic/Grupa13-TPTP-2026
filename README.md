# Stranica za promociju darivanja krvi u BiH

Ovaj projekat je rađen kao dio zadatka za predmet Tehnologije za podršku tehničkom pisanju i predstavlja web stranicu čiji je cilj podizanje svijesti o značaju darivanja krvi u Bosni i Hercegovini. Stranica omogućava korisnicima da se informišu o akcijama darivanja krvi, saznaju osnovne informacije o procesu darivanja, te se motivišu da postanu dobrovoljni davaoci.
Za izradu ovog projekta koristili smo HTML, CSS i JavaScript, bez korištenja dodatnih frameworka.
